Política de Privacidad

La aplicación Upbc recopila y utiliza datos personales, como la matrícula y la contraseña del alumno, únicamente para iniciar sesión en el sistema académico de la UPBC. Estos datos no se almacenan a menos que el alumno lo solicite explícitamente. Si el alumno decide guardar estos datos, se almacenan utilizando cifrado seguro. Además, la aplicación permite almacenar la contraseña del Wi-Fi asignada al alumno y su horario académico, siempre bajo petición del usuario y también de forma cifrada.

Los datos personales recopilados se utilizan exclusivamente dentro de la aplicación para facilitar el acceso a la información académica. Estos datos no se comparten con terceros ni se usan para otros fines.
